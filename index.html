What is CitrusHRMS?

CitrusHRMS is an Open Source Software (OSS) which can be used by small and large organization to perform various human resource functions like storing employee data, managing payroll and bonuses, hiring new recruitment, leave management, time and attendance, employee performance metrics and training records
